# DAILBERT

*Dispatches from an office run by large language models.*

A black-and-white webcomic about **Dailbert**, the engineer who builds the company's
agentic systems; **Doug**, his oblivious coworker; the pointy-haired **Boss**; and
**the Clanker** — the LLM that runs everything and appears *only* inside a surface
(a monitor, a dead screen, a night window's reflection, a projector, the sheen of a
mug, the boss's glasses). The clanker is never physically in the room.

The strips are **dated**. They begin on 2026-07-09 and march forward. Nobody in the
strip is watching the calendar. The reader might.

## How it's drawn

Every strip is **hand-authored vector ink** — no raster art, no AI image models.
A strip is pure *content* (who is present, who says what, which surface holds the
clanker); a renderer composes consistent art from a fixed set of character models.
The data model can't even *express* a clanker standing in the room — the clanker
only ever speaks through a panel's `surface`.

- `lib/art.mjs` — the character models (Dailbert, Doug, Boss), the clanker, the
  surfaces (monitor / dead-monitor / window / projector / glasses / mug), balloons.
- `build.mjs` — composes each `strips/*.json` into `strips/*.svg` and builds `index.html`.
- `strips/*.json` — one strip = content only.

```bash
node engine/seed-opening.mjs   # write the 12 opening strips (Jul 2026)
node build.mjs                 # render all strips + the gallery page
```

## How it's written (the two-agent engine)

New strips are generated by two subagents that **only know about each other a little** —
that asymmetry is the whole point. See [`world/bible.md`](world/bible.md).

- **World-Sim** advances the state of the world era by era. It privately knows the
  trajectory. It emits only *office leakage*: the handful of mundane things a normal
  worker would actually notice.
- **Joke-Writer** receives *only* that leakage and a date. It never sees the almanac
  and never learns where any of this is going. It writes oblivious office comedy.

```bash
# (run via the Workflow tool) engine/timeline.workflow.mjs
node engine/ingest.mjs <result.json>   # sanitize generated strips into strips/*.json
node build.mjs
```

## Ratings

Each strip carries an **audience score** — an integer on a **1…10¹⁰ logarithmic**
scale, read as *effective reach* (creative output is power-law). The editor assigns
them; early strips honestly sit at 1–10. Scores live in `world/ratings.json`
(decoupled from content), render as a badge + log meter on each strip, and the page
offers a **Timeline / Top rated** sort (`#top`). Re-score anytime:

```bash
node engine/seed-ratings.mjs && node build.mjs
```

## Publishing

Static site. Served by GitHub Pages from `main` / root. `.nojekyll` keeps the SVGs
untouched. Open `index.html` for the gallery.
